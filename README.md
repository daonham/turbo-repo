### Information

This is built with Next.js, Shadcn UI, Tailwind CSS, Drizzle ORM, and Better Auth.

### Tech Stack

- Next.js 15
- Shadcn UI
- Tailwind CSS v4
- Drizzle ORM
- Better Auth (Authentication)
- Upstash Redis (Cache)
- Upstash Ratelimit (Rate Limiting)
- AWS S3 (Storage)
- AWS SES (Email)
- AWS CloudFront (CDN)
- AWS Route 53 (DNS)
- AWS Certificate Manager (SSL) (CDN)
- Axiom (Logging)
- Sentry (Error Tracking)
- Tinybird (Analytics)
- Vercel (Deployment)
- Posthog (Analytics)
- Vercel AI SDK (AI)
